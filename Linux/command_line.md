# Linux Command Line

## Commands

### less

A simple file viewer:

```sh
less file_name.txt
```

| Command     | Action                                         |
|:------------|:-----------------------------------------------|
| b           | page up                                        |
| space       | page down                                      |
| G           | end of file                                    |
| 1G          | beginning of file                              |
| /characters | search the file for occurences of "characters" |
| n           | repeat the previous search (next)              |
| q           | quit                                           |

### file

Use to determine the type of of a file:

```sh
file file_name.txt
```
